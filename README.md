# GA4 Traffic Analytics Dashboard 🌐  

## 📌 Introduction  
This **Looker Studio** dashboard analyzes website traffic from **Google Analytics 4 (GA4)**, focusing on:  
1. **Channel/campaign performance** (Paid vs Organic)  
2. **Geographic engagement patterns**  
3. **Landing page conversion rates**  
4. **Year-over-Year (YOY) trend comparisons**  

---

## 📊 Core Metrics Tracked  
### **Traffic Overview**  
- 157,253 sessions (-60.7% YOY)  
- 70,084 users (-59.2% YOY)  
- 30.2% engagement rate  

### **Top Performers**  
- **Country**: United Kingdom (137,214 sessions)  
- **City**: London (32,546 sessions, 2,445 key events)  
- **Landing Page**: Contact page (427 conversions from campaign traffic)  

### **Device & Channel Data**  
- **Paid Search**: 117.7K sessions (2024 YTD)  
- **Mobile Dominance**: 76.7% of total sessions  

---

## 🎯 Project Objectives  
1. **Identify** high-converting traffic sources  
2. **Optimize** underperforming campaigns/landing pages  
3. **Track** YOY/MOM trends for strategic planning  
4. **Visualize** geographic hotspots for localized marketing  

---

## 🧹 Data Processing  
- Calculated **Engagement Rate** = (Engaged Sessions / Total Sessions)  
- Normalized **YOY Comparisons** across 8 channel groups  
- Derived **Bounce Rate Impact** by landing page  
- Aggregated **City-Level Data** for geo-targeting

## 🔍 Key Findings  
- **Opportunity**: Contact page converts at 3× higher rate than homepage  
- **Alert**: Nigeria has low bounce rate (39%) but only 3K sessions - untapped potential  
- **Issue**: Direct traffic has 81% bounce rate (needs better onboarding)  
- **Trend**: Paid Search grew 168% since 2021 (now 75% of total sessions)  

---

## 🚀 Recommended Actions  
1. **Redesign Homepage** to match contact page's conversion efficiency  
2. **Launch Geo-Targeted Ads** for Nigerian/Lagosian audiences  
3. **Create Direct-Traffic Landing Pages** to reduce bounce rate
